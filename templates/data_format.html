Fage Format

Data Format

Upload csv file, containing 3 columns: CATEGORY, WIKI_PAGE_URL, PAGE_INTERLINK

CATEGORY - can contain up to 5 categories named upon request. each category will receive a different marker style in the final analytical dashboard (can be found in the figure's legend). Notice: pages which are hyper-links of other pages from the list should be named "Interlink".

WIKI_PAGE_URL - url for the wikipedia page.

PAGE_INTERLINK - if some page is a hyper-link of other page. you should specify the page url in this column as well as the original page url under the "WIKI_PAGE_URL" column, and also make sure there is an additional row that contain only the original page url under the "WIKI_PAGE_URL" column.
